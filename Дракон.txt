2. Страница привычек
GET /habits/
Отображает список всех привычек с их основными параметрами:
Название привычки.
Описание.
Время выполнения (например, "08:00").
Сколько раз в день нужно выполнять.
Сколько дней подряд уже выполнено.
Максимальное количество дней для выполнения.

3. Страница создания привычки
POST /habits/
Позволяет создать новую привычку. Поля для заполнения:
Название привычки (обязательно).
Описание (опционально).
Время выполнения (например, "08:00").
Сколько раз в день нужно выполнять (по умолчанию 1).
Максимальное количество дней для выполнения (опционально).

4. Страница редактирования привычки
UPDATE /habits/<id>/
Позволяет изменить параметры существующей привычки по её ID. Поля для редактирования:
Название.
Описание.
Время выполнения.
Количество выполнений в день.
Максимальное количество дней.

5. Страница удаления привычки
DELETE /habits/<id>/
Удаляет привычку по её ID.

6. Страница отметки выполнения привычки
UPDATE /habits/<id>/complete/
Позволяет отметить выполнение привычки за текущий день.
Привычка отмечается как выполненная на текущую дату.
Увеличивается счетчик дней подряд.
Если привычка не выполнена, счетчик дней подряд сбрасывается.

7. Страница статистики*
GET /habits/stats/*
Отображает статистику по всем привычкам:
К конкретной привычке
Процент выполнения каждой привычки.
Количество дней подряд.
Графики прогресса (например, за последнюю неделю или месяц).
